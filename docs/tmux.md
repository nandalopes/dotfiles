# tmux

Install tmuxp `sudo pip3 install tmuxp`

- <https://tmuxp.git-pull.com/>

Layouts:

- even-horizontal: side-by-side
- even-vertical: empilhados
- main-horizontal: topo principal
  - main-pane-height: ${MAIN_PANE_HEIGHT} / NUM
- main-vertical: esquerda principal
- tiled: igualmente encaixados, dividindo 2 a 2
  - main-pane-width: ${MAIN_PANE_WIDTH} / NUM
